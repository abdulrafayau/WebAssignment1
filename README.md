SKILLSPARK is a practical tech education platform designed for developers, designers, and builders. Founded on the premise that "the best way to defend is to learn to attack," the platform offers specialized training in secure development, mobile apps, and design. 

This project was developed by Abdul Rafay as "Web Technologies â€” Assignment No. 1" for BSIT-2-B at Air University. It features a distinct cyber aesthetic UI and serves as a front-end showcase of modern web design principles.

## âœ¨ Features
* **Multi-Page Architecture:** Includes five fully structured pages: Home, Courses, About, Contact, and Register.
* **Responsive Framework:** Built utilizing Bootstrap 5.3.0 to ensure a responsive layout across desktop and mobile devices.
* **Theme Toggling:** Features a custom Light/Dark mode toggle that dynamically updates CSS variables and saves the user's preference in their browser's local storage.
* **Cyberpunk Aesthetics:** Employs a custom color palette (cyan, magenta, yellow) and specialized Google Fonts (Share Tech Mono, Orbitron, VT323) to create a tech-focused, hacker style interface.
* **Course Catalog Interface:** A structured grid layout displaying various tech courses like Secure Web Development, UI-UX, AI & Data Science, and Web Security Fundamentals.
* **Interactive Forms:** Contains styled form inputs for user registration (with a simulated password strength bar) and a secure transmission contact form.
